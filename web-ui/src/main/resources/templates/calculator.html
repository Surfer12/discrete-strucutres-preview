Quick Reference
Basic Operators:
  • AND - Logical AND (∧)
  • OR - Logical OR (∨)
  • NOT - Logical NOT (¬)
  • XOR - Exclusive OR (⊕)
Advanced Operators:
  • NAND - NOT AND (↑)
  • NOR - NOT OR (↓)
  • IMPLIES - Implication (→)
  • BICONDITIONAL - If and only if (↔)
